# IROS Paper Project Page Template

This is a skeleton project page for an IROS conference paper presentation.

## Structure

- `index.html`: Main page structure.
- `static/css/style.css`: Custom styles.
- `static/images/`: Put your images here.
- `static/videos/`: Put your videos here.
- `static/pdfs/`: Put your paper PDF and other documents here.

## How to use

1.  Replace the placeholder text in `index.html` with your paper's title, authors, and abstract.
2.  Add your own images and videos to the `static/` directory.
3.  Update the links to your paper, video, and code in `index.html`.
4.  Customize the BibTeX citation.
